Dotaku Apps

Project Date 2020
Keyword Android Studio, Kotlin
Client None
Visit github.com/anggawidiarta/Dotaku-MobileApp

The following project was made by myself when I took a mobile programming course, this project was the final project of the course I took, in this project I made a simple application that aims to provide information to people about the heroes in the video a game called Dota 2, where the information presented is what the skills of the hero are and what their backstory is or is called lore, some basic and advanced fundamental features have been implemented in this project.

This project was built using Android Studio and Kotlin as the programming language, I use Kotlin because this language is more practical than Java which has a more complicated level of complexity
